## About this project

Development of a clone of the popular Uber application that offers taxi services, using Firebase and Google Maps API.

## Libraries Used 

<ul>
<li>Firebase: Auth, Database, Storage and Messaging</li>
<li>RxJava</li>
<li>Material Design</li>
<li>Dexter</li>
<li>Google Play Services: Location and Maps</li>
<li>Geofire</li>
<li>Circle Image View</li>
<li>Glide</li>
</ul>
